# JavaScript Practice Projects

This repository contains small JavaScript projects built for practice while learning core JavaScript concepts and DOM manipulation.

Each folder represents a standalone mini-project.

## Purpose
The purpose of this repository is to:
- Practice JavaScript fundamentals
- Improve understanding of DOM manipulation and event handling
- Build small interactive projects through hands-on learning

## Projects
- **Calculator** – A basic calculator implementing arithmetic operations and user input handling
- **To-Do List** – A task management app for adding, displaying, and removing tasks
- **Weather App** – An application that fetches and displays weather data using an external API  
  *(Work in progress)*

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Most projects can be run by opening the corresponding `index.html` file in a web browser.

Some projects may require an internet connection to fetch external data (e.g. APIs).

## Notes
- These projects were built for learning purposes.
- Code may be refactored or extended as concepts improve.
- Not all projects are feature-complete.
